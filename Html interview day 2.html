HTML Interview Questions Day 2

Q.Difference between block and inline element ?

  • Block Elements
  • A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.
  • A block-level element always takes up the full width available (stretches out to the left and right as far as it can).
  • Two commonly used block elements are: <p> tag and <div> tag.

  • Inline Elements
  • An inline element does not start on a new line.
  • An inline element only takes up as much width as necessary.
  • This is a <span> element inside a paragraph

Q.Explain br tags ?

  • If you want to start a new line, you need to insert a line break with the help of the <br >. The <br > tag inserts a single carriage return or breaks in the document. This element has no end tag.

Q.Explain how many difference types of lists we have in html ?

  • There are three list types in HTML:

  • Unordered HTML List

  • An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.
  • The list items will be marked with bullets (small black circles) by default:

  • Ordered HTML List

  • An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.
  • The list items will be marked with numbers by default:

  • HTML Description Lists

  • HTML also supports description lists.
  • A description list is a list of terms, with a description of each term.
  • The <dl> tag defines the description list, the <dt> tag defines the term (name), and the <dd> tag describes each term:

Q.What are Semantic tags ?

  • A semantic element clearly describes its meaning to both the browser and the developer.
