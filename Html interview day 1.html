HTML Interview Questions Day 1

Q.Difference Between Div and Span?

  • Div is a block level element and span is inline level element.
  • Div always Start on the new line and it will take whole width of the screen and span takes the space required for it,It does not occupy unecessary width.

Q.Difference Between Class and IDs?

  • IDs and Class are selectors and are used to give name to element, using id and class we can target a particular or multiple element.
  • IDs are unique and class are not,we can apply multiple classes to any element but we cannot apply same value of any ID to difference element.In one File there should be just one id

Q.What are self closing tag?

  • In HTML, the self closing tags are the tags that does not require closing tag to get terminated.
  • EX:- img tag,br tag,hr tag,etc

Q.How many types of headings are there ? How to decide which to use where

  • In HTML, there are 6 types of heading tags,from h1 to h6.
  • h1 stands for bigger one and h6 stands for smaller one.
